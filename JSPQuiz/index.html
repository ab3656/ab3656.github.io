How are JSP pages related to servlets?
When a browser requests a page with a .jsp file the container reads the request. Using the JSP compiler, the container converts the JSP into a servlet class that The JSP file is compiled only when the page is first requested, or when the JSP file has been changed. Otherwise, the previously compiled JSP servlet class is re-used, making subsequent responses much quicker. Finally, the generated JspPage servlet class is invoked to handle the browser request.
How are JSP pages related to HTML?
JSP pages can have plain html code embedded or generate dynamic HTML code on the go.
Give an example of a JSP scriptlet and show how it will look in the JSP servlet.
Example of scriptlet: 
                <% int x=10;
                int y=40;
                int z = x+y;
                out.println("Sum of x and y is " + z);
                %>
The scriptlet above will be compiled to the following in JSP servlet: 
                int x=10;
                int y=40;
                int z = x+y;
                out.println("Sum of x and y is " + z);
Give an example of a JSP declaration and show how it will look in the JSP servlet
Example of JSP declaration: 
                <%! int count = 0; %>
The declaration above will be compiled to the following in JSP servlet: 
                int count = 0

Explain how an EL expression is evaluated.
Expression -> ${variable}
The container checks page scope for an attribute named variable, and if not found keeps on checking it in request scope, session scope and application scope and if still not found, it ignores the expression.
Explain how servlet attributes are involved in EL expressions.
Suppose the EL expression: ${sessionScope.name} is in the JSP page. The EL expression will search for objects in servlet attributes in the following order
  1. pageScope
  2. requestScope
  3. sessionScope
  4. applicationScope
If the expression is stored in session scope as follows: 
            String name = "Ameet";
            session.setAttribute("name", name);
The EL expression ${sessionScope.name} will give us "Ameet"
Explain how servlets and JSPs use request dispatch to interact.
Servlets and JSP use request dispatch to set and get attributes to other JSPs or servlets.
