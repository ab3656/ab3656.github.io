Can post requests be bookmarked? What are the problems?
In the POST method the data is passed in the HTTP request body so it can't be bookmarked because bookmarking only saves the URL and not the request body.
What is the purpose of request dispatching?
The purpose of request dispatching is to internally pass the requst processing to another local resource or to tell the client browser to issue another http request to a specified URL.
What is the difference between redirect and request dispatch?
Request dispatch is used to pass the request to another resource for further processing within the same server, another resource could be any servlet, jsp page any kind of file. Redirect is used to redirect client request to some other location for further processing ,the new location is available on different server or different context

During request dispatch browser is completely unaware of servlet forward and hence the URL in browser address bar will remain unchanged whereas during redirect this request is visible in browser as a new request.

Request dispatch is also called server side redirect whereas redirect is called client side redirect.

With request dispatch, request and response objects are passed to destination servlet whereas in redirect, only http message is sent.
What is an attribute ?
Attributes are name value pair objects bound into one of the three servlet API objects; HttpServletRequest, HttpSession, ServletContext
What is the difference between attributes and parameters?
Attributes are server variables that exists within a specified scope in application, session or requests whereas parameters are string values passed from client to server.

Attributes are set in a servlet for the same request and is accessible from JSP and can be any object including string. Parameters are set and passed from client to server and can only return strings.
What does it mean to say that http is stateless? Give an example of a stateful protocol.
HTTP is a stateless protocol, which means that after a web server responds to a request, there is no “memory” of the request or who sent it. Every resource that is accessed via HTTP is a single request with no threaded connection between them. FTP(File Transfer Protocol) is an example stateful protocol. It saves the status and session information.

How long does a session last?
Sessins last as long as the declared value of session-timeout in web.xml. getMaxInactiveInterval method tells how long sessions last.
What is a cookie, and how long does a cookie last?
Temporary cookies are deleted when the browser is closed. Permanent cookies has a max age set so it lasts until that time.
